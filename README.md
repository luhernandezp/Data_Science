# Ciencia de Datos.
Este repositorio contiene los resultados, análisis y entregables desarrollados en el marco de la prueba técnica para el rol de Ciencia de Datos.

# Resultados prueba tecnica 1.

Como parte de la entrega, se incluye un informe en formato PDF con el análisis detallado de los datos. Adicionalmente, se adjunta el archivo fuente `.qmd`, utilizado para la generación reproducible del documento, junto con un script en R que contiene el código completo empleado en el desarrollo del análisis.
